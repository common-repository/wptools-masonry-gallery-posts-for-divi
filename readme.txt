== Description ==

## Images & Blogs Masonry Gallery Divi Modules

https://www.youtube.com/watch?v=P5vycmTdOC8

Create masonry gallery grid for images/blogs using our easy-to-use divi modules.

This plugin offers 2 divi modules,

* **Masonry Image Gallery Module** - Creates masonry gallery of WordPress media library images.
* **Masonry Posts Module** - Creates masonry gallery of custom post types like blogs, projects etc.

## Masonry Image Gallery Divi Module

https://www.youtube.com/watch?v=SmSkyZHSECQ

Use images from media library and create a masonry gallery.

* Manually select/upload images OR query images using custom filter
* Ability to categorize images
* Create popover gallery on image click
* Create overlay on image mouse-over.
* Tons of styling options

**Documentation :** [https://wptools.app/how-to/how-to-create-masonry-image-gallery-in-divi/](https://wptools.app/how-to/how-to-create-masonry-image-gallery-in-divi/?utm_source=wp-org&utm_medium=plugin-page&utm_campaign=divi-masonry&utm_content=description)

## Masonry Posts Divi Module

https://www.youtube.com/watch?v=0seSu7oREww

Use blog data from [custom post types](https://wordpress.org/support/article/post-types/#custom-post-types) to create masonry gallery.

* Create blog items list using custom filter
* Setup pagination
* Create overlay on image mouse-over.
* Tons of styling options

**Documentation :** [https://wptools.app/how-to/how-to-create-masonry-blog-gallery-in-divi/](https://wptools.app/how-to/how-to-create-masonry-blog-gallery-in-divi/?utm_source=wp-org&utm_medium=plugin-page&utm_campaign=divi-masonry&utm_content=description)

== Installation ==

How to install the `WPTools Masonry Gallery & Posts For Divi` plugin and get it working.

1. Upload `wptools-masonry-gallery-posts-for-divi` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
